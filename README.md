## 🔬 Descripción de Componentes

### 1. Arquitectura RISC-V (MP1)

En esta etapa nos enfocamos en el diseño de un núcleo procesador basado en el estándar abierto **RISC-V**, implementando su arquitectura y sistema mínimo para ejecución de instrucciones.

### 2. Control de Periféricos y E/S (MP2)
Implementación de sistemas robustos utilizando:
* **Interrupciones y Timers:** Para una gestión eficiente del tiempo de CPU.
* **Protocolos de comunicación:** Manejo de periféricos vía **SPI** e **I2C**.
* **Almacenamiento:** Uso de memoria **EEPROM** para persistencia de datos.

### 3. Sistemas de Tiempo Real (Extra)

Integración de un sistema final bajo el kernel de **FreeRTOS**, permitiendo la gestión de tareas concurrentes y recursos críticos en tiempo real.
